# Expense Tracker

A full-stack personal finance application that allows users to record, view, filter, and analyze their expenses.

## Tech Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **ORM**: Prisma
- **Validation**: Zod
- **API Client**: Axios

## Core Requirements

1. Create new expense with amount, category, description, and date
2. View list of expenses
3. Filter expenses by category
4. Sort expenses by date (newest first)
5. Display total amount of visible expenses
6. Handle duplicate expense creation via idempotency

## Deployment Platforms

- **Frontend**: Vercel / Netlify
- **Backend**: Render / Railway / Heroku
- **Database**: Neon (PostgreSQL) / Supabase
